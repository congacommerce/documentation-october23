Conga Digital Commerce June '23

Upgrading the Partner Commerce Template from September '22 to June '23

This guide is designed to step you through the process of upgrading partner commerce template to consume SDK from September '22 release to June '23 release.

Table Of Contents

  • Updating the libraries
  • Updating npmrc file
  • Update request quote form component file

Updating the libraries

You must point these packages to the latest version:

  • @congacommerce/core ^2302.x.x

  • @congacommerce/ecommerce ^2302.x.x

  • @congacommerce/elements ^2302.x.x

After updating these packages, verify if your package.json under dependencies reflects the latest version as below:


"@congacommerce/core": "^2302.x.x",

"@congacommerce/ecommerce": "^2302.x.x",

"@congacommerce/elements": "^2302.x.x",

...

Updating npmrc file

To install SDK libraries as npm packages, you must update .npmrc file to specify the github repository path where these packages are hosted.

@congacommerce:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken={{GITHUB AUTH TOKEN}}

As this is a public github repository with read permissions, you don't need any access to install these packages.

Update Request Quote Form component.

In June '23 release, you must update shipToChange, billToChange and primaryContactChange methods in RequestQuoteForm component to handle PrimaryContact, ShipToAccount, BillToAccount field values when set to null in request quote page.

1.) billToChange method:-

Before Change

billToChange() {
    ...
      this.onQuoteUpdate.emit(this.quote);
  }

After Change

 billToChange() {
    if (this.quote.BillToAccountId) {
      ...
    } else {
      this.quote.BillToAccount = null;
      this.billToAccount$= null;
      this.onQuoteUpdate.emit(this.quote);
    }
  }

2.) shipToChange method:-

Before Change

shipToChange() {
    ...
      this.onQuoteUpdate.emit(this.quote);
  }

After Change

 shipToChange() {
    if (this.quote.BillToAccountId) {
      ...
    } else {
      this.quote.ShipToAccount = null;
      this.shipToAccount$= null;
      this.onQuoteUpdate.emit(this.quote);
    }
  }

3.) primaryContactChange method:-

Before Change

primaryContactChange() {
    ...
  }

After Change

primaryContactChange() {
  if (this.contactId) {
    ...
  } else {
    this.quote.Primary_Contact = null;
    this.onQuoteUpdate.emit(this.quote);
  }
}
